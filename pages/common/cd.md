# cd      &#x6C49;&#x5B57;

> Change the current working directory.

- Go to the given directory:

`cd {{path/to/directory}}`

- Go to home directory of current user:

`cd`

- Go up to the parent of the current directory:

`cd ..`

- Go to the previously chosen directory:

`cd -`
